Indian Railways PNR Enquiry API

Request:

http://pnrapi.alagu.net/<10-digit-pnr-number>

Sample Response:

 
"status" : "OK",
"data" : { 
          "train_name" : "ERNAKULAM EXP",
          "train_number" : "12677",
          "from" :  {"name" : "Bangalore City" , "code":"SBC"},
          "to" :  {"name" : "Coimbatore Jn." , "code":"CBE"},
          "alight" : {"name" : "Coimbatore Jn." , "code":"CBE"},
          "board" : {"name" : "Bangalore City" , "code":"SBC"},

Options:

Supports JSONP. Example: http://pnrapi.alagu.net/<10-digit-pnr-number>?jsonp=yourcallbackfunction

Status Codes:

CodeWhat it means
OKAll iz well. You should get a response
TIMEOUTThere was a server timeout. Server timeout is 1 second.
INVALIDInvalid PNR Number. It doesn't exist.
